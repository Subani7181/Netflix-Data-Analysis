## About the Project
The goal of this project is to analyze Netflix‚Äôs content library and visualize patterns in content type, genre, ratings, release trends, and geographical distribution using **Power BI**.

---

## Dataset
- **File:** [Netflix Data.xlsx](https://github.com/Subani7181/Netflix-Data-Analysis/blob/main/Netflix%20Data.xlsx)  
- **Size:** 5,800+ titles.  
- **Content Types:** Movies and TV Shows.  
- **Key Columns:** Title, Type, Genre, Country, Release Year, Runtime, IMDb Rating.  

---

## Project Workflow
1. **Data Collection** ‚Äì Gathered Netflix data from web sources.  
2. **Data Cleaning** ‚Äì Removed duplicates, corrected data types, handled missing values.  
3. **Data Modeling** ‚Äì Built a many-to-one relationship model using a unique ID column.  
4. **DAX Calculations** ‚Äì Created measures for KPIs such as total content, runtime, average rating.  
5. **Dashboard Creation** ‚Äì Designed an interactive Power BI dashboard using charts, maps, and slicers.
